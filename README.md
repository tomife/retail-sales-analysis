# Retail-Sales-Analysis

### Project Overview
A retail sales analysis project containing dashboards, KPIs, and summary reports for buyers, categories, managers, and monthly performance.

### Data Sources

The primary dataset used for this analysis is the "Retail-Sales-Analysis-Excel" file

### Tools

- Microsoft Excel
  - Data analysis and creating dashboard
    

### Exploratory Data Analysis

EDA involved exploring the Retail-Sales-Analysis to answer key questions,such as
- Total sales revenue
- Top buyer of the year
- Most Profitable Suburb
- Top manager of the year

  ### Data Analysis
  - Identify trends and patterns in the dataset "Retail-Sales-Analysis"
    
  Visualization: Built interactive visuals including KPIcards,line chart,bar chart,timeline and slicer to present insights clearly.

 ### Results/Findings
  - Top 5 buyers
  - Top 5 selling categories
  - Top 5 best selling days
  - Top 5 suburb groups
  - Top 5 best selling months
  - Top 5 best selling managers
